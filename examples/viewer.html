プリセットビューアー

プリセットの状態ルール、アクション、継続時間による昇格を確認し、同じ入力に対する判定結果をブラウザ上で比較できます。

使用する定義と比較対象（Active / Comparison）

現在の定義を評価し、必要に応じて別の定義と結果を比較できます。

入力（Input）

フォームを編集すると、内部の入力 JSON も同時に更新されます。

JSONを確認する（Developer View）

この JSON が実際にエンジンへ入力されます。
